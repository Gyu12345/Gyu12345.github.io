C++基本语法之输入输出

输入和输出相对于C++的其他函数来说是十分简单的.

首先是第一种输入输出cin 和cout:

cin和cout,需要搭配前置"using namespace std;"和头文件"#inlcude"

具体格式如下:

#include

using namespace std;

int main()

{

int a;//定义一个变量

cin>>a;

cout<< a;

//另外值得一提得是，在没有using namespace std;的时候可以使用std::cin和std::cout代替cin和cout

}

第二种便是C中的scanf 和 printf

这种相对cin和cout来说优点不少

格式是这样的：

scanf("%?",&a); printf("%?",&a)

这里的问好便是格式控制符

主要是%d整数 %c字符 %lf小数具体的可以自己上网查询(doge)
